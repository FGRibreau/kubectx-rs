# kubectx

Fastest switch between Kubernetes contexts 🏎.

![kubectx](/docs/kubectx.gif)

### Setup

```bash
npm i kubectx -g
```

### Usage

```sh
➜  / kubectx
--Select kubectl context (Use arrow keys)
  image-charts-production
❯ killbug-development
  minikube
  ouest-france-development
```
